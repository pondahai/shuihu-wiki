# 張清

**別名**:沒羽箭

**總出現次數**:292,**出場回數**:30 回

### 生平概述

[[張清]]，別號「沒羽箭」，原為東昌府守將，後歸順梁山泊。他擅長使用石子打人，百發百中，曾連打梁山泊十五員大將，展現高超的飛石功夫。歸順後，他被任命為馬軍八虎騎兼先鋒使之一，成為梁山重要的軍事指揮官。

在梁山接受招安後，[[張清]]跟隨[[宋江]]南征北戰。在征遼過程中，他多次以石子擊傷或擊落遼國將領，如阿里奇、兀顏統軍等，立下赫赫戰功。其勇猛表現深受[[宋江]]賞識，並被列入十驃騎將序列。

征討田虎時，[[張清]]受吳用計策派遣，假扮醫士進入襄垣城，成功接近[[瓊英]]。經過一番較量與接觸，[[張清]]與[[瓊英]]結為夫妻，並裏應外合誅殺[[鄔梨]][[田虎]]。此役後，[[張清]]與[[瓊英]]一同受到朝廷封賞，復還舊職。

在後續征討王慶與方臘的過程中，[[張清]]繼續發揮其飛石特長，但在獨松關之戰中，因槍搏[[厲天閏]]不中而卡在松樹上，最終被[[厲天閏]]還槍戳死。戰後，朝廷追封其為忠武郎。

*   連打梁山泊十五員大將，包括[[徐寧]][[韓滔]][[彭玘]][[宣贊]][[呼延灼]][[劉唐]][[楊志]][[朱仝]][[雷橫]][[關勝]][[董平]][[索超]]。（第70回）

*   用石子打中嵩州都監[[周信]]鼻凹，使其翻身落馬，並與[[龔旺]][[丁得孫]]合力將其戳死。（第77回）
*   與[[董平]]一同巡哨濟州，在鳳尾坡截擊[[王文德]]，用石子打中其盔頂使其逃走。（第78回）
*   活捉水中扭打的[[韓存保]]；用石子打中[[梅展]]額角流血；馬被[[張開]]射中後跳至岸上步戰，後奪馬退回。（第79回）
*   在濟州城下與[[宋江]][[盧俊義]][[吳用]][[公孫勝]]並馬向[[高俅]]聲喏；奉[[吳用]]之令埋伏，用石子打中[[丘岳]]面門。（第80回）
*   密雲縣交戰時，用石子正中[[阿里奇]]左眼使其落馬，功績簿上標寫為第一功。（第83回）
*   檀州城下，用石子擦過[[洞仙侍郎]]耳根；遼國救兵來時，用石子打中[[耶律國寶]]面部使其落馬。（第83回）
*   在玉田縣陣前被遼將[[天山勇]]用弩箭射中咽喉，翻身落馬。（第84回）
*   被分派為東方甲乙木青袍軍馬戰遼國土星主將陣的副將之一；取石子亂打，傷及[[兀顏光|兀顏統軍]]身畔牙將，打中[[兀顏光|兀顏統軍]]面門及頭臉，最終用槍結果其性命。（第89回）
*   病癒後向[[宋江]]請示，並在[[安道全]]欲解釋「瓊矢鏃」含義時，以目視之使安道全微笑不開口。（第93回）
*   領兵與[[林沖]][[索超]][[徐寧]]出征，用石子打中[[竺敬]]鼻凹使其落馬；用石子打中[[山士奇]]腦後頭盔使其伏鞍而走。（第94回）
*   領輕騎五百，星夜出關馳往[[衛州]]，接取[[公孫勝]]並趕到軍前。（第96回）
*   衝殺過來，趕上[[孫琪]]，手起一鎗搠中後心將其撞下馬來。（第96回）
*   聽報[[瓊英]]飛石傷人後，被[[吳用]]選中假扮醫士全羽之弟全羽，進入[[襄垣城]]。（第98回）
*   在演武廳與[[瓊英]]比試武藝，接住其飛石並對打，打得雪片般落下。（第98回）

### 人物關係

**配偶：**
*   [[瓊英]]：妻子。兩人結為夫妻，並裏應外合誅殺[[鄔梨]][[田虎]]。

**上司/主公：**
*   [[宋江]]：梁山泊首領，後為宋朝先鋒。[[張清]]始終效忠於[[宋江]]，並在其麾下立下多項戰功。
*   [[吳用]]：軍師。[[張清]]多次執行[[吳用]]的命令，如埋伏官軍、假扮醫士進入襄垣城等計策。

**盟友/同伴：**
*   [[龔旺]][[丁得孫]]：部下或同伴，常一起行動，如哨路、衝截官軍等。
*   [[董平]]：戰友。兩人常一同執行任務，如巡哨濟州、攻擊遼兵等。
*   [[安道全]]：醫生與同伴。[[張清]]生病時與[[安道全]]前往高平療治，病癒後一同參見[[宋江]]。
*   [[公孫勝]]：戰友。[[張清]]負責接取[[公孫勝]]，並與其一同趕到軍前。
*   [[解珍]][[解寶]]：戰友。[[張清]]命令二人分守四門，後來也與他們共同行動。
*   [[葉清]]：同伴。[[張清]]與[[葉清]]共同執行任務，如擒捕[[田虎]]、押解賊首等。
*   [[張青]][[孫二娘]][[唐斌]][[文仲容]][[崔埜]][[耿恭]][[曹正]][[薛永]][[李忠]][[朱富]][[時遷]][[白勝]]：夥伴。在攻佔襄垣後，與眾人一起殺戮偽官及賊徒。

**敵人/對手：**
*   [[周信]][[王文德]][[韓存保]][[梅展]][[張開]][[丘岳]][[周昂]][[王煥]][[徐京]][[楊溫]][[李從吉]]：北宋官員或將領，曾與[[張清]]交戰。
*   [[阿里奇]][[洞仙侍郎]][[耶律國寶]][[天山勇]][[兀顏光|兀顏統軍]]：遼國將領，曾被[[張清]]擊敗或殺害。
*   [[竺敬]][[山士奇]]：田虎部將，曾被[[張清]]用石子打傷或打退。
*   [[鄔梨]]：田虎部將，[[瓊英]]之父，被[[張清]]與[[瓊英]]裏應外合鴆死。
*   [[唐昌]]：田虎部將，被[[張清]]刺殺。
*   [[糜貹]]：王慶部將，曾撥落[[張清]]的石子。
*   [[劉以敬]][[上官義]]：王慶部將，被[[張清]]和[[瓊英]]隔開兵器。
*   [[厲天閏]]：方臘部將，最終殺死了[[張清]]。
